About Portal

ALMS is a multi-component Learning Management System engineered for library automation. Built on the latest Java 23 LTS and JavaFX 23, it delivers high-performance desktop applications for both administrators and end-users. Our architecture leverages a hybrid database approach with PostgreSQL for transactional integrity and MongoDB for flexible analytics.

From barcode scanning integration to automated email notifications via Jakarta Mail, ALMS provides a robust, secure, and scalable solution. Whether deployed on Windows, macOS, or Linux, our system ensures operational resilience and data sovereignty through advanced encryption and modular design.

Help & Installation Guide

Deployment Protocols

Execute the following sequence to initialize the software:

  • Select the appropriate binary for your operating environment.
  • Secure the installer to your local drive.
  • .MSI Executables: Initiate the setup wizard and authorize installation.
  • .JAR Archives: Verify Java Runtime Environment (JRE 8+) integrity, then execute.
  • Initialize the application and authenticate via secure login.

System Requirements

  • Operating System: Windows 10+, macOS 10.14+, Linux (Ubuntu 18.04+)
  • Java Runtime: JRE 8+ (JDK 23 recommended for development)
  • RAM: 4 GB minimum (8 GB recommended for JavaFX)
  • Storage: 150 MB (Admin) / 100 MB (User) free space
  • Internet: Required for setup & integrations; offline mode available

Common Issues and Troubleshooting

  • Execution Failure: Verify JRE installation and version compatibility.
  • Network Error: Inspect firewall configurations for blocked connections.
  • Resource Allocation: Confirm available RAM and disk space meet minimums.
  • Support: Contact the technical division for advanced diagnostics.
